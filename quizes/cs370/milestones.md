What was the key innovation of the IBM System/360 introduced in 1964?

- It was the first family of machines with the same language but different sizes and power
- It was the first computer to use integrated circuits
- It was the first personal computer sold in stores
- It was the first computer to use a bus architecture

---

What problem did the System/360 solve for IBM customers?

- Customers had incompatible machines (7094 and 1401) and wanted compatibility
- Customers wanted faster processors than the 7094 offered
- Customers needed more memory than existing machines could provide
- Customers wanted machines that could run multiple operating systems

---

What concept did the System/360 pioneer that we now call concurrency?

- Multiprogramming
- Virtual memory
- Time-sharing
- Parallel processing

---

What does VLSI stand for in the context of computer generations?

- Very Large Scale Integration
- Virtual Logic System Interface
- Variable Length Signal Input
- Vector Logic Sequence Indexing

---

What made personal computers possible according to the milestones?

- VLSI chips became cheap enough with millions of transistors
- The invention of the bus architecture
- The development of MS-DOS
- The creation of the Apple II

---

How were the first personal computers like the Intel 8080 kit sold?

- As kits with circuit board, chips, power supply, and floppy disk (no software)
- As fully assembled machines with pre-installed operating systems
- As software packages that users could install on existing hardware
- As rental units that could be returned after use

---

What was significant about the Apple II compared to earlier personal computers?

- It was one of the first fully built (not kit) personal computers
- It was the first computer to use integrated circuits
- It was the first computer to use a graphical user interface
- It was the first computer to connect to the internet

---

What did IBM do with the design of the IBM Personal Computer that started the PC industry?

- Made the design available for $49 so other companies could make plug-in boards
- Released all patents to the public domain
- Licensed the design exclusively to Microsoft
- Open-sourced the hardware specifications

---

What happened to OS/2, the successor to MS-DOS created by Microsoft and IBM?

- It failed, and Microsoft's Windows became dominant instead
- It became the standard operating system for all IBM computers
- It was renamed to Windows NT
- It was acquired by Apple and integrated into macOS

---

What is the x86 architecture named after?

- The Intel 80386 and its predecessors (8086, 80186, 80286)
- The 86 transistors on the first chip
- The 86 wires in the bus architecture
- The 86 instructions in the instruction set
