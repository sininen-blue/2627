# High level programming languages

Scratch is an example of "high level programming languages"

high level just means that it's "removed" or "abstracted" from what the computer wants to do

this means that they don't work the way a computer actually works, but it's easier to read, has more convenience features, and has specific tools available that aren't how computers work but models how humans think well enough that it doesn't matter
